# Plan: Create Missing Pet Seeder

## Goal
Create a database seeder that populates the `animals` table with sample missing pet records for testing.

## Current State
- Missing pets are stored in the `animals` table using:
  - `is_missing = true` (boolean flag)
  - `missing_since` (date field)
  - `last_seen_location` (text field)
  - `contact_info` (text field)
- The `animals` table already exists with these fields

## Implementation Plan

### Step 1: Create MissingPetSeeder
- Location: `database/seeders/MissingPetSeeder.php`
- Create sample missing pet records with fields:
  - animal_type, species, name, gender, color, breed
  - is_missing = true
  - missing_since = random dates within last 30 days
  - last_seen_location = random barangay locations
  - contact_info = contact details

### Step 2: Register in DatabaseSeeder
- Add `MissingPetSeeder::class` to the `run()` method in `database/seeders/DatabaseSeeder.php`

### Step 3: Run the seeder
- Command: `php artisan db:seed --class=MissingPetSeeder`

## Sample Data Structure
```php
// Example record:
[
```
