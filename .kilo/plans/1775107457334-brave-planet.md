# Rabies Geomap UI Redesign - 12-Column Map-Dominant Layout

## Goal
Restructure the Rabies Geomap page into a clean 12-column layout where the sidebar is purely for navigation and the map dominates the main content area with floating overlay elements.

## Files to Modify

### 1. `resources/views/layouts/admin.blade.php` (Layout Shell)
**Change**: Enforce a proper 12-column CSS grid structure.

- Keep the fixed sidebar at `width: 256px` (approx 2/12 of typical screen)
- Keep `main { margin-left: 256px }` as the effective 10-column content area
- No structural changes needed — the sidebar + main margin already implements the 2/10 split
- **No changes to sidebar content** — it already only contains navigation

### 2. `resources/views/city-vet/rabies-geomap.blade.php` (Primary View)
**Major restructuring**:

#### REMOVE:
- The 4-column stats grid above the map (`grid-cols-4` row with Total Cases, Affected Barangays, Critical Areas, Reporting Period)
- The `bg-white rounded-xl border` card wrapper around the map
- The toolbar bar with `border-b` between header and map
- Any `p-6` padding on the outer content div

#### ADD:
- **Map fills the entire content area**: `height: calc(100vh - 64px)` (64px = topbar height), no padding wrapper
- **Floating stats bar** (top-right inside map): Small pill-shaped stat indicators (`Total: X | Affected: Y | Critical: Z`) using `position: absolute; top: 12px; right: 12px; z-index: 1000`
- **Floating filter bar** (top-left inside map): Year/Month/Week dropdowns + Reset button as a compact floating toolbar with backdrop-blur
- **Legend stays bottom-left** (already there, keep as-is)
- **Map subtitle** integrated into the floating filter bar area

#### Layout structure:
```
<div class="relative w-full" style="height: calc(100vh - 64px);">
  <!-- Map fills entire area -->
  <div id="geomap" style="height: 100%; width: 100%;"></div>

  <!-- Floating: Filter bar (top-left) -->
  <div class="absolute top-3 left-3 z-[60] flex items-center gap-2 backdrop-blur bg-white/90 rounded-lg px-3 py-2 shadow-sm border border-slate-200/60">
    [Title] + [Year] [Month] [Week] [Reset]
  </div>

  <!-- Floating: Stats pills (top-right) -->
  <div class="absolute top-3 right-3 z-[60] flex items-center gap-2">
    <div class="backdrop-blur bg-white/90 rounded-lg px-3 py-1.5 shadow-sm border border-slate-200/60 text-xs">
      <span class="text-slate-400">Total</span> <span class="font-bold text-slate-800">24</span>
    </div>
    <div class="backdrop-blur bg-white/90 rounded-lg px-3 py-1.5 shadow-sm border border-slate-200/60 text-xs">
      <span class="text-slate-400">Affected</span> <span class="font-bold text-amber-600">8</span>
    </div>
    <div class="backdrop-blur bg-white/90 rounded-lg px-3 py-1.5 shadow-sm border border-slate-200/60 text-xs">
      <span class="text-slate-400">Critical</span> <span class="font-bold text-red-600">2</span>
    </div>
  </div>

  <!-- Floating: Legend (bottom-left) - existing component -->
  <x-map-legend />

  <!-- Floating: Filter tags (below filter bar, only when active) -->
  <div id="filterSummary" class="absolute top-14 left-3 z-[60] ...">
    [filter tags]
  </div>
</div>
```

### 3. `resources/views/components/map-legend.blade.php` (Legend Component)
**Minor changes**:
- Move position from `bottom-left` to `bottom-right` to match user spec ("bottom-right or bottom-left floating card")
- Reduce width slightly to `160px`
- No other structural changes needed

### 4. `resources/views/dashboard/rabies-geomap.blade.php` (Duplicate View)
**Mirror the same changes** as city-vet view. Only difference: the AJAX URL uses `baseUrl + '/data'` instead of named route.

## z-index Hierarchy
| Element | z-index |
|---------|---------|
| Map (Leaflet tiles) | 1 |
| Leaflet controls | 10 |
| Floating filter bar | 60 |
| Floating stats pills | 60 |
| Filter dropdown menus | 1000 |
| Floating legend | 1000 |
| Leaflet zoom controls | 1001 |

## Design Tokens
- Floating elements: `backdrop-blur bg-white/90 border border-slate-200/60 shadow-sm rounded-lg`
- Stats pills: compact `text-xs`, icon optional, color-coded numbers
- Filter buttons: same `.filter-btn` style but smaller (`text-[11px] px-2 py-1`)
- No outer padding on the map container — map edge-to-edge within the content area

## Spacing
- Content area: `p-0` (no padding — map fills the space)
- Floating elements: `12px` from edges
- Top bar height: `64px` (h-16), map fills `calc(100vh - 64px)`
- Mobile: sidebar collapses, map fills full width

## What NOT to change
- `routes/web.php` — no route changes
- `CityVetController.php` — no controller changes
- `map-legend.blade.php` structure — only position change
- JavaScript logic — all filter/map/data logic stays the same
